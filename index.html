My Projects

Bayesian Optimization

Bayesian optimization (BO) is a powerful framework to quickly search for near-optimal solutions and accelerate discovery process. It is widely applied in material design, hyper-parameter tuning, and combinatorial optimization. This project aims to solve fundamental problems of BO when using Gaussian process as a surrogate model. The project includes following modules:

  • Development of a Bayesian optimization approach to handle discrete input data. [paper]
  • Missing data imputation for refining historical data. [paper]
  • Selection of the optimal strategy in cost restriction settings using a multi-armed bandits algorithm. [paper]

    Applied Machine Learning in Material Science

    Machine learning techniques provide great support to reduce human workload and encourage automation, especially for tasks which require high precision. In this project, the target is to predict high throughput materials, e.g., efficient solar cells and thermal conductors, and accelerate discovery process of novel materials, e.g., new crystalline materials. This project has two aims:

  • Batch experiments recommendation for Metal-Organic Frameworks (MOFs).
  • Uncertainty quantification for high-quality material prediction.
    •

    Time-series prediction and optimization

    Continuous-time prediction is a compelling and critical concept in real-time applications, particularly within the financial sector. This project involves forecasting dynamic data streams as they evolve, enabling rapid decision-making in environments where timing is paramount. By leveraging advanced models and real-time data processing, the project aim to support applications such as high-frequency trading, risk management, and portfolio optimization, driving efficiency and informed strategies in fast-paced markets. Two current focused topics in this project are

  • Accelerating large portfolio optimization.
  • Stock prices forecasting.
